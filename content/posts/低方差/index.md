对于特征的选择，可以直接基于方差来判断。低方差的特征意味着该特征的所有样本值几乎相同，对预测影响极小，可以将其去掉。

```python
import numpy as np
```


```python
# 生成特征
x1 = np.random.randn(100)
print(np.var(x1))
```

    1.08004809622322



```python
x2 = np.random.normal(5,0.1,100)
print(np.var(x2))
```

    0.009174853126249195



```python
# 将特征组合为矩阵X
X = np.vstack((x1, x2)).T
print(X.shape)
```

    (100, 2)



```python
# 特征选择：低方差过滤法
from sklearn.feature_selection import VarianceThreshold
vt = VarianceThreshold(threshold=0.01)
```


```python
x_filtered = vt.fit_transform(X)
print(x_filtered.shape)
print(x_filtered)
```
